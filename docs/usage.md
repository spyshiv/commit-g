# Usage 🛠️

### Step 1: Stage Your Changes

```bash
git add .
```

### Step 2: Run Commit-G

```bash
commitg
```

### Step 3: Follow the Prompts

- ✅ Accept the suggestion

- ✏️ Edit the message

- 🔄 Regenerate it

- ❌ Cancel

Commit-G makes it interactive and easy to maintain a clean commit history.
